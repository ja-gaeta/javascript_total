Construtores

O JavaScript possui o tipo objeto, que pode ser visto como uma coleção de propriedades e métodos de acesso, podendo inclusive conter outros objetos, permitindo a construção de estruturas extremamente complexas, mas não possui o tipo formal class e, embora o ECMAScript 2015 (ES6) tenha introduzido classes na linguagem, elas são, na verdade, simplificações para as heranças baseadas nos protótipos.

Que fique claro, então, que o JavaScript não é uma linguagem orientada a objetos, como Java ou C++ e suas classes fornecem apenas uma maneira mais simples de criar objetos e lidar com herança.

Para criarmos vários objetos com a mesma estrutura, usamos uma função construtora de objeto, simplesmente chamada constructor. Uma vez definido o constructor, as instâncias do objeto são criadas com a palavra-reservada new.
